1) What is the core difference between a class and an interface?

A class can have fields, constructors, and implemented methods, and objects can be created from it
An interface defines a contract that classes must follow

2) Can interfaces have fields? If yes, what are the implicit modifiers for them?

Yes. All interface fields are implicitly public, static, and final

3) Can a class implement multiple interfaces? Explain why this is useful

Yes. A class can implement multiple interfaces
This is useful because one class can have several behaviors at the same time
